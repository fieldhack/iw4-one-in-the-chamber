# One in the Chamber for IW4

## Installation
Copy `one_in_the_chamber.gsc` to /userraw/scripts in your server directory.

#### OPTIONAL:

Additionally copy `weapon_drops.gsc` to disable players dropping ammo. 

## Setup
### Recommended Settings
Gamemode Free-for-all:

```g_gametype "dm"```

Three total lives:

```scr_dm_numlives "3"```

Ensure one shot kills:

```scr_player_maxhealth 10```

Disable perks:

```scr_game_perks 0```

Disable killstreaks:

```scr_game_hardpoints "0"```
